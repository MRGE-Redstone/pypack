# Contributing

Contributions are always welcome, but please follow the guidelines carefully.

## How to Contribute

- [Fork this repository](https://github.com/MRGE-Redstone/mcpypack/fork)
- Make your changes
- Open a pull request with your changes.

## Guidelines
- Do not submit AI-generated code.
- Always use type annotations.
- Variable, function, class, or method names, as well as comments should be in English.
